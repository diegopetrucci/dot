# My dotfiles

This is the public subset of my private [chezmoi](https://www.chezmoi.io) setup. I sync selected files here from `~/.local/share/chezmoi` whenever I get around to it, so this repo is representative rather than exhaustive.

## What's in here

- AI agents:
  - [Pi](./.pi)
  - [Codex](./.codex)
  - [OpenCode](./.config/opencode) 
- AI skills:
  - [Skills](https://github.com/diegopetrucci/ai-agents-skills)
  - [Pi extensions](https://github.com/diegopetrucci/pi-extensions)
- [Atuin](./.config/atuin): shell history/search.
- [Chezmoi package data](./.local/share/chezmoi/.chezmoidata/packages.yaml): the package inventory that helps keep machines aligned.
- [Ghostty](./.config/ghostty): terminal config and theme overrides.
- [Git](./.gitconfig) and [ignore rules](./.config/git).
- [Neovim](./.config/nvim): my editor config.
- [Powerlevel10k](./.p10k.zsh): prompt config for Zsh.
- [Scripts](./.local/bin/scripts): small utilities I actually use.
- [Tmux](./.config/tmux): less central now that I mostly live in Ghostty, but still useful for persistent sessions.
- [Zed](./.config/zed): editor settings and theme placeholders.
- [Zsh](./.zshrc) and [modular shell config](./.config/zsh): aliases, completions, lazy-loading, and shell startup tweaks.
